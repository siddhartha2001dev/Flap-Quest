🐦 Flap Quest – A Flappy Bird Clone

Welcome to Flap Quest, a simple and addictive arcade-style game inspired by the classic Flappy Bird. Built entirely using HTML, CSS, and JavaScript, this project is a fun way to explore DOM manipulation, animation, and game logic using web technologies.

Game Prototype Design (Figma) : https://www.figma.com/design/g5xWhGaRDAjpDbFn5X50JM/FLAP-QUEST-FINAL-DESIGN?node-id=0-1&t=yhvfsf74DxZWf0MZ-1


🎮 Gameplay
The objective is simple:
Tap or click to keep your bird flying and navigate through endless pipes without crashing. The further you go, the higher your score!

🚀 Features
Smooth and responsive gameplay

Simple 2D graphics with CSS styling

Score tracking system

Collision detection

Restart feature after game over

Lightweight and runs on any modern browser

🛠️ Technologies Used
HTML5 – Structure of the game

CSS3 – Styling and animations

JavaScript (Vanilla) – Game logic, interactivity, and DOM updates
